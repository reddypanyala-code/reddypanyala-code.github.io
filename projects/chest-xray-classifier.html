Machine Learning-Based Chest X-ray Classifier

Research Project | Guidance: Prof. Ashish Nimgaonkar

Context

This project focused on large-scale medical imaging analysis using the NIH Chest X-ray dataset.

Problem

Clinical image review is data-intensive. The challenge was to build a model pipeline capable of estimating disease probability from chest X-ray images at scale.

Methods

  • Worked with a dataset of approximately 112,000 images.
  • Prepared data and structured model inputs for training and evaluation.
  • Developed a machine learning classifier for probability-based prediction output.

Outcome

Built a functioning classifier that produced disease-probability estimates from patient X-rays, strengthening experience in data-driven model development and validation thinking.

My Role

Executed model development workflow, from dataset handling through probability output interpretation, under faculty mentorship.
